Use Case 1 — Produce a Video

uv run pipeline produce \
  --url "https://youtube.com/watch?v=…" \   # required
  --locale zh-TW \                    # zh-TW | ja | es-MX
  --niche parenting \                 # auto-detected from channels.toml; "none" to skip metadata
  --voice tim-zhtw \                  # optional; auto-selects per locale from voices/registry.json
  --start-from tts \                 # resume from a specific stage
  --skip-review \                    # bypass human review gate (auto-apply proofread fixes)
  --subtitles                        # burn subtitles into final video

Scene has black/dark backgroundFix storyboard bg_prompt or prompt: avoid dark-concept keywords (pool, mirror, underwater, panels). Delete compose/scenes/sXX_*.{png,mp4} and the matching image_cache/*.png. Dark-retry logic in image.py handles future runs automatically.--start-from compose

Storyboard Editing Commands

uv run pipeline storyboard show                                    # list all scenes
uv run pipeline storyboard show --scene scene_003                  # one scene detail
uv run pipeline storyboard recordings --voice tim-zhtw              # recording status per scene
uv run pipeline storyboard set scene_003 narration="新文字"           # edit field in-place
uv run pipeline storyboard set scene_003 pause_after_sec=2.5        # timing tweak

Image Generation (generate-image skill)

# Step 1 — always draft first
python3 ~/.claude/bin/gen-image.py "場景描述 — title card, dark background, zh-TW text" \
  --tier draft --size landscape --output output/projects/<ID>/compose/assets/scene_003.png

# Step 2 — approved? regenerate at production quality
python3 ~/.claude/bin/gen-image.py "same prompt" \
  --tier production --size landscape --output output/projects/<ID>/compose/assets/scene_003.png

# Fixed seed — reproducible result across tiers
