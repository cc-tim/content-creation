## Core Concept (Do Not Translate)

The TED talk introduces the "Running Leap" metaphor: children approach challenges by *backing up* first to build confidence. What parents read as avoidance or laziness is actually preparation. The child's intrinsic motivation is already present — interrupting the backing-up process (by pushing or pleading) either produces resentful compliance or erodes confidence further.

This video is **NOT a translation** of the talk. It is an original zh-TW narrative that uses the same insight, recast through examples Taiwanese parents recognize immediately.

## Scene Structure (24 scenes × ~22s = ~8.8 min)

### Hook (S1–S3) — ~1 min
Build tension; withhold the reframe until it is earned.

| Scene | Purpose | Narration (zh-TW) |
|-------|---------|-------------------|
| S1 | Set familiar scene | 台灣家庭的餐桌旁，孩子盯著課本，緩緩把它推到一旁。父母站在身後，忍不住開口：「快點，你還有好多題沒寫！」 |
| S2 | Hook question | 你有沒有這樣的時刻？孩子明明有能力，就是不肯動手。你越催，他越不動。你開始懷疑：是懶？是逃避？還是根本不在乎？ |
| S3 | Deepen tension (NOT the reframe) | 但如果你的催促，正好打斷了他最重要的一步呢？ |

### The Concept (S4–S6) — ~1.5 min
Introduce the running-leap metaphor cleanly before any examples.

| Scene | Purpose | Narration |
|-------|---------|-----------|
| S4 | Stream metaphor (visual) | 想像你走在山路上，遇到一條小溪。要跨過去，你會怎麼做？ |
| S5 | Concept build | 溪流越寬，你退得越遠——不是因為你怕，是因為你需要足夠的助跑距離。孩子面對挑戰，也是一樣的道理。 |
| S6 | Name the concept | 這個動作，叫做助跑起跳。是孩子天生內建的動力機制。每一次他們「退後」，都是在找到起跑點，準備飛越眼前的挑戰。 |

### Example 1 — Piano (S7–S9) — ~1.5 min
*Most zh-TW resonant — promoted to first. Hits the 補習費 nerve.*

| Scene | Purpose | Narration |
|-------|---------|-----------|
| S7 | The refusal | 小欣坐在鋼琴旁的地板上，雙臂交叉，說什麼都不肯練習。 |
| S8 | Parent's frustration | 媽媽在廚房裡深吸一口氣。補習費那麼貴，請了老師，買了琴——你就坐在那裡？ |
| S9 | The reveal | 但那天傍晚，媽媽路過客廳，聽到了琴聲。小欣一個人坐在琴椅上，輕輕地、一遍又一遍地練著那首曲子。 |

### Example 2 — Math Frog-Leap (S10–S12) — ~1.5 min
*Lighter, more playful — provides tonal relief after the emotional piano story.*

| Scene | Purpose | Narration |
|-------|---------|-----------|
| S10 | The overwhelm | 另一個孩子，面對一張數學練習卷，突然睜大眼睛說：「我要爆炸了！爆炸了！砰！」 |
| S11 | The frog-leap routine | 然後她從椅子上跳下來，像一隻青蛙一樣，繞著桌子跳了一圈，又爬回椅子上。 |
| S12 | Reframe | 她不是在鬧，她在充電。一次青蛙跳，換回一道數學題。做完全部題目的時間？其實沒有比坐著硬撐更長。 |

### Example 3 — Classroom Refusal (S13–S15) — ~1.5 min
*zh-TW school context — the shame of not raising your hand is universally familiar.*

| Scene | Purpose | Narration |
|-------|---------|-----------|
| S13 | The withdrawal | 課堂上，老師問了一個問題。小明知道答案，但他縮進椅背，頭微微低下，沒有舉手。 |
| S14 | Backing up at home | 那天晚上回到房間，他反覆練習那個答案，小聲地自己說了好幾遍。 |
| S15 | The leap | 隔天，老師又問了類似的問題。這一次，小明的手舉起來了。 |

### Science + Validation (S16) — ~30s
*One scene; bridges examples to action while emotional temperature is still warm.*

| Scene | Purpose | Narration |
|-------|---------|-----------|
| S16 | Validate the pattern | 三個孩子，三種不同的方式，做的是同一件事——在挑戰面前先退一步，讓信心追上來，再起跳。信心，是助跑跑出來的。 |

### 3 Steps — Actionable (S17–S19) — ~1.5 min

| Scene | Purpose | Narration |
|-------|---------|-----------|
| S17 | 觀察 | 第一步：觀察。找找看，他有沒有偷瞄那個大滑梯？孩子在「逃避」的同時，往往還是在盯著目標看。 |
| S18 | 等待 | 第二步：等待。你不需要成為他的動力，他的動力已經在了。你的工作，是不要擋在他的助跑路線上。 |
| S19 | 說出來 | 第三步：說出來。當他完成的時候，告訴他你看見了什麼——「你知道自己需要什麼，你找到方法了。」 |

### Visual Payoff (S20) — ~22s
*New scene — show the magic sentence landing, not just stated.*

| Scene | Purpose | Narration |
|-------|---------|-----------|
| S20 | The sentence landing | 這句話的力量，不是讚美，是鏡子——讓孩子看見自己的能力是真實的，不是靠外力給的。說完那句話，你會看見他的臉，從一點點不確定，變成一點點相信自己。 |

### The Risk (S21) — ~22s
*Brief but necessary — consequences of interrupting the backing-up.*

| Scene | Purpose | Narration |
|-------|---------|-----------|
| S21 | What goes wrong | 如果在他還沒準備好時強迫起跳——最好的結果是勉強服從，最壞的是讓他開始覺得自己很爛。信心越低，下次需要的助跑距離就越長。我們以為在推他，其實是在讓那條溪流，變得更寬。 |

### The Reframe — Earned (S22) — ~22s
*Now finally give the S3 answer. Earned after all examples.*

| Scene | Purpose | Narration |
|-------|---------|-----------|
| S22 | Deliver the reframe | 孩子不是在逃避。他正在準備起飛。 |

### Quiet Closing (S23–S24) — ~1 min

| Scene | Purpose | Narration |
|-------|---------|-----------|
| S23 | Trust as the gift | 父母能給孩子最好的禮物，不是永遠在旁邊催促，而是讓他知道：就算他退後幾步，你還是在那裡，你相信他。 |
| S24 | Final image + call to action | 下次孩子退後，先停下來想一想——他是在逃避，還是在助跑？ |

### Key overrides vs. defaults
- **No source clips**: storyboard `clip` fields must be null/absent on all scenes; only `rich_slide` or `image` visual type
- **Image tier**: `premium` (OpenAI) — patch rich_slide.py before running compose
- **TTS**: edge-tts `zh-TW-YunJheNeural` (male narrator tone matches the reflective style)
- **Scene count**: 24 scenes, target ~22s average duration

### Stage flow
1. `acquire` — download transcript only (no clip extraction needed)
2. `analyze` — extract story structure / knowledge graph
3. `direct` — write custom storyboard (do NOT use auto-generated one; hand-craft from this spec)
4. Human review gate → proofread
5. `tts` — generate narration audio per scene
6. `compose` — render all-image video (patch image tier first)

### Direct stage override
Skip the LLM-driven `direct` stage entirely. Instead:
1. Run `acquire` + `analyze` to create the project dir and knowledge graph
2. Hand-write `storyboard.json` directly from this spec (narration strings, theme block, `rich_slide` visual type per scene)
3. Run `produce --project-id <ID> --start-from tts` to pick up from there

This avoids LLM drift from the designed scene structure. The narration strings in this spec are canonical — use them verbatim as each scene's `narration` field.
